Estudando HTML

Nossos estudos começaram falando a respeito de um item muito sério que é a organização. Isso porque a linguagem HTML é responsavel apenas pela marcação dos elementos na janela do navegador. O HTML não cria nada, ele manda buscar ou utiliza as letrinhas que você digita.

Parágrafo

Para colocarmos parágrafod, usamos "p" e "/p". Para fazer uma quebra de linha, matendo o mesmo parágrafo, usamos o elemento "br". Temos ainda parágrafos especiais, denominados títulos, cabeçalhos ou headings em que são usados elementos de "h1" a "h6" e seus respectivos fechamentos de "/h1" a "/h6".

Imagens

Uma imagem aparece na janela do navegador pelo uso do "img" Esse elemento possui o atributo "src", responsavel plea localização da imagem. Para trabalhar melhor com a imagem, você pode colocá-la dentro das tags de marcação de parágrafo.
